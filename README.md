# universepdf
UniversePDF is a free online tools website that helps users work with
PDFs, images, and file utilities easily and securely.

## Available Tools

UniversePDF currently offers a growing collection of tools, including:

### PDF Tools
- PDF Splitter
- PDF Compressor
- PDF Merger
- Merge PDF & Images
- Reorder PDF Pages
- Rotate PDF
- Watermark PDF
- PDF to Text

### Image Tools
- Images to PDF
- PDF to JPEG
- JPEG to PNG
- WEBP to JPEG
- HEIC to JPEG
- Image Compressor
- PNG Compressor

### Utility Tools
- QR Code Generator
- QR Code Reader
- Create ZIP File
- Extract ZIP File

More tools are added gradually based on user needs.

## Purpose
The goal of UniversePDF is to provide simple, fast, and accessible tools
without requiring login or complex setup.
